#Project Overview:
This project focuses on building a multi-page Healthcare Analytics Dashboard to analyze patient demographics, admission trends, department performance, and financial outcomes including insurance analytics.
The objective was to transform raw healthcare data into meaningful business insights to support data-driven decision-making in hospital management.

#Data Visualization Tool: Power BI  Data Preparation: Excel / CSV Techniques Applied: Data Cleaning, Calculated Fields, Filters & Parameters, KPI Indicators, Trend Analysis

#Business Impact: 
The dashboard enables hospital management to monitor operational performance, optimize resource allocation, evaluate insurance partnerships, and improve overall patient care efficiency through data-driven insights.
